# Ecommerce Project

## Overview

This is a full-stack MERN ecommerce application built with React, Node.js, Express, MongoDB, and Tailwind CSS. It includes:

- User authentication and authorization
- Product browsing and search
- Shopping cart and checkout flow
- Order management
- Admin panel for product and order management
- Cloudinary image upload support
- PayPal payment integration

## Tech Stack

- Client: React 19, Vite, Tailwind CSS, Redux Toolkit, React Router
- Server: Node.js, Express, MongoDB, Mongoose
- Authentication: JWT, cookies
- Payments: PayPal REST SDK
- File upload: multer, Cloudinary

## Folder Structure

- `Client/` - frontend React application
- `Server/` - backend Express API and database connection
